# โยงคู่ไม่ให้ชน

เกม HTML/CSS/JavaScript แบบไม่ใช้ไลบรารีภายนอก เหมาะสำหรับนำขึ้น GitHub Pages

## จุดเด่น

- สุ่มโจทย์ใหม่ทุกครั้ง
- สร้างคู่แบบ non-crossing matching ก่อน จึงมีคำตอบอย่างน้อยหนึ่งชุดเสมอ
- ตรวจเส้นออกนอกวงกลม เส้นชนกัน และเส้นผ่านรูปอื่น
- 5 ระดับความยาก ตั้งแต่ 2 ถึง 6 คู่
- รองรับเมาส์และหน้าจอสัมผัส
- มีคำใบ้ ย้อนกลับ จับเวลา ดาว และเสียงเอฟเฟกต์จาก Web Audio API
- ไม่ต้องดาวน์โหลดไฟล์เสียงหรือไลบรารีเพิ่ม

## วิธีใช้งานบนเครื่อง

เปิด `index.html` ในเบราว์เซอร์ได้ทันที

## นำขึ้น GitHub Pages

1. สร้าง repository ใหม่ใน GitHub
2. อัปโหลด `index.html`, `style.css`, `script.js`
3. ไปที่ Settings > Pages
4. เลือก Deploy from a branch
5. เลือก branch `main` และโฟลเดอร์ `/root`
6. กด Save แล้วรอระบบเผยแพร่

## หลักการสุ่มที่รับประกันว่ามีคำตอบ

ระบบจะสุ่มจุดรอบวงกลม แล้วสร้าง perfect matching แบบ non-crossing บนลำดับจุดดังกล่าว จากนั้นจึงกำหนดสัญลักษณ์ให้ปลายเส้นแต่ละคู่ มีการตรวจซ้ำว่าคำตอบไม่ตัดกัน ไม่ผ่านจุดอื่น และมีระยะห่างเพียงพอ หากตัวอย่างสุ่มไม่ผ่านจะสุ่มใหม่ และมี fallback ที่แก้ได้แน่นอน
