# One Stroke Adventure

เกมลากเส้นครั้งเดียวสำหรับเล่นบนเว็บไซต์ รองรับเมาส์ ปากกา และหน้าจอสัมผัส

## วิธีใช้งาน
1. แตกไฟล์ ZIP
2. เปิด `index.html` ในเว็บเบราว์เซอร์
3. เมื่อต้องการเผยแพร่บน GitHub Pages ให้อัปโหลดไฟล์ทั้งหมดขึ้น repository แล้วเปิด Pages

## กติกา
- เริ่มลากจากจุดใดจุดหนึ่ง
- ลากต่อเนื่องโดยไม่ยกมือ
- ใช้เส้นทุกเส้นให้ครบ
- ห้ามลากเส้นเดิมซ้ำ

## โครงสร้าง
- `index.html` โครงหน้าเว็บ
- `style.css` รูปแบบและ responsive design
- `game.js` ข้อมูลด่าน ระบบตรวจเส้น เสียง เอฟเฟกต์ และความคืบหน้า

## หลักการออกแบบด่าน
รูปแต่ละด่านถูกแทนด้วยกราฟ โดยจุดคือ vertex และเส้นคือ edge
ด่านที่ลากได้ในครั้งเดียวต้องมีจุดดีกรีคี่ 0 หรือ 2 จุด (Euler circuit / Euler trail)
